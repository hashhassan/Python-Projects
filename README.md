# Python Projects

A collection of simple Python projects built during university semesters.

---

## 1. YT Video Downloader

A simple YouTube video downloader with a basic GUI where you paste a
YouTube link and download the video directly to your computer.

### Libraries Used
- pytube — for downloading YouTube videos
- tkinter — for the graphical user interface (GUI)

### How to Run
1. Make sure Python is installed on your computer
2. Install pytube:
   pip install pytube
3. tkinter comes built-in with Python, no installation needed
4. Run the file:
   python yt_downloader.py

---

## 2. Alarm Clock

A simple alarm clock application with a basic GUI where you set a time
and it plays a sound when the alarm goes off.

### Libraries Used
- tkinter — for the graphical user interface (GUI)
- winsound — for playing the alarm sound

### How to Run
1. Make sure Python is installed on your computer
2. Both tkinter and winsound come built-in with Python,
   no installation needed
3. Run the file:
   python alarm_clock.py

### Note
winsound only works on Windows, so this app will not run on Mac or Linux.
